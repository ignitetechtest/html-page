Welcome to Svelte

A radical new approach to building user interfaces. Svelte shifts the work into a compile step rather than doing the work in the browser.


No Virtual DOM

Svelte doesn't use a Virtual DOM. Instead, it updates the DOM directly when your state changes.

Truly Reactive

No more complex state management libraries. Svelte brings reactivity to JavaScript itself.

Less Code

Write less code and build boilerplate-free components using languages you already know — HTML, CSS and JavaScript.
